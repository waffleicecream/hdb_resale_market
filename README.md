# PropertyMinBrothers — HDB Resale Market Dashboard

Singapore HDB resale market analysis and flat valuation tool.

- **Market Analysis**: Choropleth map with town-level price and transaction metrics
- **Amenities Comparison**: Compare up to 3 HDB blocks by proximity to MRT, schools, hawkers, etc.
- **Flat Valuation**: ML-powered price prediction using a Random Forest model with conformal intervals
